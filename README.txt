Run main.py to open the project (flask)

Required python package:
 -	requests, flask

The main page contains a search bar where user can search for tracks, albums or artists. The result page contains both the search bar and the results. User can search again or click on specific buttons to view more information about the results. User can change the scope of the search on the search bar, e.g. if user search only for tracks, then only track results are shown. User can click on search button on each result to search for that results specifically. For each artist, user can click on detail button to view more information about the artist. For each track, user can click on the comment button to see comments related to the track on YouTube. User can also click on each track to listen to the track on Spotify.
